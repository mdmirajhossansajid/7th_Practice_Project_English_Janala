📖 Project Overview

English Janala is a simple and interactive web-based project designed to help users learn and practice English vocabulary. It provides an easy interface where users can explore words, meanings, and improve their language skills.



This project is part of my practice work to strengthen frontend development skills.

🚀 Features
📚 Vocabulary learning system
🔍 Word meaning display
🎯 Simple and user-friendly UI
⚡ Fast and responsive design
🌐 Runs directly in the browser (no installation needed)
🛠️ Technologies Used
HTML5
CSS3
JavaScript
🔗 Live Demo
👉 https://mdmirajhossansajid.github.io/7th_Practice_Project_English_Janala/
🎯 Learning Purpose


This project helped me to:

Improve JavaScript DOM manipulation

Practice UI/UX design basics

Build small interactive web applications
